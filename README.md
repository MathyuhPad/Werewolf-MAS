# Werewolf-MAS
Small-scale recreation of a Werewolf game using MAS.


This project was our first experience with Multi-Agent Systems, as well as with the language itself, which likely explains some of the suboptimal design choices and implementation decisions. The project was developed with the Gama tool in mind, and follows some conventions and constraints specific to that environment.

This contains our implementation of a voting and relations system. The number of players that will be given each role can be changed in the first lines of the code, which allows for experimentation.

The code can be run and tested by opening Gama, opening the model and launching the simulation.
